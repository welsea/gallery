# Gallery
J2EE项目开发与设计大作业

## 功能介绍

* 游客/普通用户：
    *  搜索艺术作品
    *  查看详情介绍页
    *  查看其他用户分享内容。
* 普通用户
   *  在游客的基础上，可以在分享页分享自己的评论/图片
   *  查看自己分享全部内容
* 管理员用户
   * 管理用户
   * 管理用户分享内容
   * 编辑/添加数据

## 主要技术
* JSP
* JDBC
* Servlet
* JavaScript
* 基础数据库知识
* 使用Bootstrap4框架
